## Whatsapp link generator (Chrome extension)

Generates a `wa.me` WhatsApp link with a prefilled message that always includes the current website (active tab URL).

### Install (developer mode)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder

### Usage

1. Click the extension icon
2. Set your DDI and WhatsApp number (saved between sessions)
3. Edit the message (the website part updates automatically as you browse)
4. Click **Generate**, then **Copy** or **Open**
